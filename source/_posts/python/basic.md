## 流程控制


### if

```python
x = 1
if x < 0:
  print('x less than zero')
elif x == 0:
  print('x equil zero')
else
  print('x great than zero')
```


### while

```python
x, b = 0, 1
while a < 10:
  print(a)
  a, b = b, a+b
```


### for

```python
words = ['Hello', 'world']
for w in words:
  print(w, len(w))

for i in range(5)
  print(i)

for i in range(len(words))
  print(i)
else
  print('not break')
```
