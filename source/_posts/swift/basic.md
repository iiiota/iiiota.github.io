> Swift语言是类型安全的语言。


# 基础


## 变量

必须先声明，并且有值才可使用。

- 声明

```swift
var min: Int    // 变量
let max: Int    // 常量

// 类型推测
var min = 10
let max = 0.1

// 指定类型
var min: Int = 10
let max: Int = 20

// 多变量声明
var min = 10; let max = 20
```

- 注释

```swift
// 单行注释

/*
多行注释，可嵌套
*/
```
